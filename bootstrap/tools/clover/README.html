Clover.NET Version 1.2 - 1.1.2053 (15-August-2005)

Welcome to an update to Clover.NET 1.2. This update addresses the following issues since the previous Clover.NET 1.2 update.

  • Improved Visual Studio integration. In some Visual Studio installations, Clover.NET would fail with a Null Reference exception. This has been corrected
  • The operation of the keyfile option has been fixed and the keyfile location is now propoerly quoted by the Clover.NET NAnt tasks

If you are upgrading from a version prior to 1.2, this release includes significant new functionality including:

  • New CloverSolution command line tool for Clovering Visual Studio Solutions from the command line.
  • New <solution> NAnt task support allowing the use of Clover.NET in continuos integration environments which use the <solution> task as the main build mechanism.
  • Major performance improvements in the parsing and instrumentation phases of Clovering.
  • New options for ordering the coverage results in HTML reports
  • Fixed a number of VB.Net parsing and instrmentation problems.

Visual Studio Plugin Installation

To install the Clover.NET Visual Studio plugin, please ensure you install the Microsoft VSIP interop assemblies beforehand and stop any Visual Studio instances you have running.

NAnt 0.85 users

A recent change to NAnt after Release Candidate 3 is not backward compatible and breaks the Clover.NET tasks. When this code is release in RC4, we will issue an updated version of the Clover.NET NAnt tasks.
