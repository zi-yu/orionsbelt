Orionsbelt Language Pack
------------------------

The Language Pack inclues the Orionsbelt language files. These resources are in XML files that can me validated with LanguageValidator. Use this validator if you intend to create other localization files for our game.

Sections to translate:
1) Orion's Belt Resources
2) Orion's Belt Wiki (Manual)
